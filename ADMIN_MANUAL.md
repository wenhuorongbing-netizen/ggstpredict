# 管理员操作手册 (ADMIN MANUAL)

作为 **OVERSEER**（系统管理员），你拥有控制赛事、结算积分和管理新玩家准入的最高权限。

## 1. 如何进入管理员面板
- 在登录界面输入权限账号（默认为 `admin`）登录。
- 进入 Dashboard 后，手动在浏览器地址栏将 URL 更改为 `/admin` 进入控制台。

## 2. 邀请码管理 (Access Control)
为了防止恶意机器人刷号，所有新玩家注册都需要唯一的“邀请码”。
- **当前版本**：管理员需通过后端直接在数据库生成邀请码。
- **操作方法 (服务器终端)**：
  ```bash
  # 进入项目目录运行 Prisma Studio 查看/修改数据
  npx prisma studio
  ```
  然后在浏览器打开 `localhost:5555`，进入 `InviteCode` 表，手动点击 "Add Record"，输入一个自定义代码（如 `GGST-BETA-001`），并将 `used` 设为 `false`。
  *(未来版本可在此面板添加自动生成 API)*

## 3. 批量智能部署 (Smart Deploy)
告别繁琐的一对一输入，快速创建整个晚上的对阵表。
- 在 `OVERSEER PANEL` 顶部的多行文本框中，按行输入比赛。
- **严格格式**：`选手A vs 选手B`
  *例如：*
  ```text
  Sol vs Ky
  May vs Ramlethal
  Zato-1 vs Anji
  ```
- **快捷标签 (Chips)**：系统会自动记录你最近输入过的选手名字。点击下方的标签，可以快速将名字追加到文本框中。
- 点击 **LAUNCH MATCHES**，系统会自动创建多场状态为 `LET'S ROCK` 的比赛。

## 4. 赛事结算 (SLASH!)
当比赛结束时，你需要立刻在系统内进行结算，以便积分发放到赢家账户。
- 在 `ACTIVE OPERATIONS` 列表中找到对应的比赛。
- 点击 **P1 WIN** (左侧胜) 或 **P2 WIN** (右侧胜)。
- **⚠️ 危险操作**：系统会弹出二次确认框。一旦确认，奖池积分（₩）将被立刻按照比例瓜分并写入玩家数据库，比赛状态变更为 `SLASH!`（无法再被修改）。
