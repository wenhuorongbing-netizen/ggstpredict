# 赏金猎人指南 (BOUNTY HUNTER GUIDE)

## 1. 动态下注限额 (Dynamic Limits)
* **小组赛 (Group Stage)**: 管理员设定固定最高限额（默认 300 W$）。
* **淘汰赛 (Knockout Stage)**: 按你的当前总资产的百分比（默认 50%）计算最高限额，但享有绝对保底（默认 200 W$）。

## 2. RC取消 (Roman Cancel)
锁盘前可撤回下注，系统扣除 5% 手续费。

## 3. 连击奖励与破招 (Combo & Counter Hit)
* **连胜奖励**: 预测2连胜结算收益 +5%，5连胜 +15%。猜错断连。
* **破招大赏**: 若双方赔率悬殊超过 9:1，且冷门方获胜，系统额外为赢家奖池注入 2000 积分。

## 4. 胜者抽水与全局狂热 (Tax & Tension)
* **胜者抽水**: 赢家的纯利润将被系统扣除 5% 作为手续费，防止通胀。
* **全服狂热**: 全服累计下注满 50,000 分，触发一次全服免抽水（Tax = 0）。
